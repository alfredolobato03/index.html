HTML

(Voltar)

O html (hiper text markup language) surgiu com a ideia de um sistema de hipertexto na internet.

É uma linguagem para estruturar uma página web; pode ser estruturada em parágrafos, em uma lista com marcadores ou usando imagens e tabelas.

Criado em 1991 pelo britânico Tim Berners-Lee , foi projetado para interligar instituições de pesquisa próxima e compartilhar documentos com facilidade.

COMO FUNCIONA O HTML

(Voltar)

O html funciona de maneira muito simples, você ira através dele, definir o que cada elemento representa dentro do seu site; e o navegador irá ler e interpretar essas marcações para que ele possa exibir o conteúdo de maneira correta.

Cada elemento html possui uma função específica, e você irá definir qual função ele irá ter dentro do seu site para que possa atender as suas necessidaes.

Como se fosse um livro, imagine que cada palavra terá uma função dentro do mesmo; você terá que definir se ela vai ser um título, um parágrafo, um subtítulo, uma imagem, um link, etc. E assim o navegador irá ler e executar da maneira correta para seu usuário.

TAGS

(Voltar)

São usadas para informar ao navegador a estrutura do site, ou seja, quando se escreve um código html, as tags serão interpretadas pelo navegador, produzindo assim a estrutura e o conteúdo visual da página.

Existem diversas tagas assim como os elementos e o exemplo do livro. Existem tags para escrever um específico título; um parágrafo ou até mesmo criar vários itens personalizados no site.

CURIOSIDADE

(Voltar)

O primeiro site da história foi criado por Tim Berners-Lee, considerado hoje como o pai da internet. Foi ao ar em dezembro de 1990 e você pode ter acesso clicando no link da página.

Html é uma linguagem de marcação e não programação, porque descreve a estrutura de um site semanticamente, juntamente com dicas de apresentação. Ela é usada escrevendo código em um arquivo de texto que é lido por um navegador e transformado em documento web visual.
